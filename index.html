FAQ

Frequently asked questions

What meetings does it work with?

Any meeting platform: Zoom, Google Meet, Microsoft Teams, Slack Huddles, FaceTime, Discord. Captures system audio, so it works with any app that produces sound.

Does it work on Intel Macs?

Yes, but Apple Silicon (M1/M2/M3/M4) is strongly recommended for real-time transcription performance. Whisper uses Metal GPU acceleration on Apple Silicon.

What languages does it support?

All 99 languages supported by whisper.cpp. Best accuracy with English. Set your language in Settings or per-session via MCP.

How large is the speech model?

Depends on the model you choose. Base (~150 MB) for speed, Large-v3 (~3 GB) for accuracy. Downloaded once on first launch.

Does it record video?

No. Mindflow captures audio only. No screen recording, no camera.

What happens when I stop recording?

The transcript saves as a Markdown file and a JSONL file to ~/.local/share/mindflow/transcripts/. You own the files.

What AI assistants does it work with?

Any MCP-compatible assistant: Claude Code, Cursor, Windsurf, and any tool that supports the Model Context Protocol.

Is it open source?

The speech recognition engine (whisper.cpp) is open source. The Mindflow app itself is proprietary, currently free during beta.

What will it cost after beta?

$39 one-time after beta. Lifetime updates included. The features you get during beta are the features you keep.

Why does macOS ask me to confirm the app is safe?

Mindflow is distributed outside the Mac App Store. Right-click the app and choose Open on first launch. This is standard for independent Mac software.
