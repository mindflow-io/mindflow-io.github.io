# Mindflow v0.5.4

**Released:** 2026-05-04

## Fixes
- "Report a Bug" button now ALWAYS opens a prefilled issue with the diagnostic body, even when the recent log is too long to fit in the URL. Previously, oversize bodies fell back to a blank issue form (with the body silently parked on the clipboard). The new flow truncates the older log lines, keeps the recent tail + environment header in the URL, and notes the truncation inline so you can paste the full clipboard dump if needed.

No functional changes outside the bug-report flow.
