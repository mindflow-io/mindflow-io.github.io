# Meetflow v0.2.26

**Released:** 2026-04-27

## What's new

- **Settings → "Models" tab** (renamed from "Transcription"). The tab still contains the same model picker, language selector, and transcription quality controls — just labelled more honestly for what it is. No setting changes; nothing to re-configure.

## Under the hood

- This is the first release where existing users keep their Screen Recording and Microphone permissions automatically. v0.2.25 introduced the stable signing identity that makes this possible; v0.2.26 is the validation that the next update over a v0.2.25 install retains TCC grants without prompting.
