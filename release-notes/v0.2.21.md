# Meetflow v0.2.21

**Released:** 2026-04-16

## What's new

- **Significantly improved transcription accuracy.** Whisper now uses beam search (beam=5) with temperature fallback, tuned silence detection, and a natural-language vocabulary prompt. In testing, proper noun recognition jumped from ~75% to ~90%+ on the remote audio channel.
- **Microphone audio is now clearer.** Apple's Voice Processing I/O is enabled on the mic input, giving you automatic gain control, noise suppression, and echo cancellation — your own voice comes through much better in transcripts.
- **Fewer hallucinated phrases.** A built-in filter catches the most common whisper hallucinations ("thanks for watching", "[Music]", etc.) and drops them before they reach your transcript.
- **Faster live updates.** Audio chunks are now 15 seconds instead of 25, so you see new transcript lines sooner during a meeting.
- **Parakeet final-pass transcription (preview).** After a meeting ends, Meetflow can now run a second, higher-accuracy pass using NVIDIA's Parakeet model via the FluidAudio engine. This is a preview feature — session audio saving will be enabled in a future release to fully activate it.
- **Fuzzy vocabulary correction.** Your custom dictionary terms are now fuzzy-matched against the transcript after each meeting. Near-misses like "Xperio" are auto-corrected to "Experio" if the term is in your vocabulary list.

## Fixes

- **Fixed silent recording failures.** If whisper can't process audio (e.g., a missing model or bad configuration), the recording card now shows a clear error instead of silently producing an empty transcript.
- **Fixed VAD model download.** The voice activity detection model download was pointing at a moved URL and silently failing. It now downloads correctly from the new location.
- **Whisper flag validation on startup.** The app now validates that all whisper command-line flags are recognized before recording starts. If a flag is invalid (e.g., after a whisper.cpp update), you'll see an error immediately instead of getting empty transcripts.

## Under the hood

- Default English model changed from medium.en to large-v3-turbo for better accuracy at similar speed.
- Audio loudness is now normalized to -20 dBFS before whisper processes it, giving consistent results regardless of input volume.
- Multi-channel system audio picks the louder channel instead of averaging, which helps when one channel is silent.
- Removed legacy CLI installer scripts (install.sh, start.sh) that were superseded by the app bundle.
