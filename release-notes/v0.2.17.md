# Meetflow v0.2.17

**Released:** 2026-04-13

## Highlights

- **Auto-updater reliability fix.** Earlier builds constructed the DMG download URL from the version number (`Meetflow-0.2.17.dmg`) but only the `Meetflow-latest.dmg` URL was published. Every auto-update attempt 404'd and saved the GitHub Pages "Page not found" page as the DMG. 0.2.17 switches the updater to always fetch `Meetflow-latest.dmg`, and release.sh now also publishes versioned DMGs as a rescue path for users stuck on buggy older builds.
- **Slash command brand scrub.** Installed `/meetflow` slash command no longer leaks implementation names or points at the stale `~/.meetflow/` path.
- **MCP tool description cleanup.** Removed residual references to passive "Claude mentions" detection from `get_new_transcript` so the tool doesn't read like it supports voice commands (it does not).
- **Release flow overhaul.** `scripts/release.sh` now skips the force-install of `/Applications/Meetflow.app` by default, so every release exercises the in-app auto-updater path instead of bypassing it. Pass `--install` to seed a fresh machine.
